**Best Project Award in DSC 106 Fall 2025 (150+ students)**

An interactive scrollytelling data story built with **D3.js** and **Scrollama** that travels back in time through key historical eras to examine how **colonial extraction, conflict, and industrialization** shaped global CO₂ emissions over time (1850–2014).
